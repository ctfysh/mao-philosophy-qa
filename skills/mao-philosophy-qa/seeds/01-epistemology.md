# 类别 A：认知与学习（认识论）

- **种子问题数**：12
- **对应哲学支柱**：认识论（实践与认识的辩证关系）
- **适用场景**：学习方法困惑、知识转化困难、判断力不足、经验系统化

---

## 设计说明

认知与学习是每个人日常面对的高频场景——从"读书用不上"到"判断力不够"，从"学了就忘"到"直觉不可靠"。此类种子问题的核心逻辑是：**用毛选认识论的四大工具（感性→理性飞跃、实践检验、矛盾特殊性、调查权）把模糊的学习焦虑转化为可操作的认识论命题**，帮助用户从"学了多少"转向"为什么学和怎么用"。

---

## 种子问题表

| # | 种子问题 | 核心原理 | 追问方向 |
|---|---------|---------|---------|
| 1 | 怎样才算真正"懂"了一件事？ | 感性→理性飞跃，实践是真理标准 | 你'懂'了之后能做什么以前做不到的事？ |
| | | **扩展**：这个问题直接触及认识论的核心——"懂"不是一个静态的存档状态，而是一个动态的实践能力。许多人用"我记得"来替代"我理解"，但毛选认识论追问的是：你能用这个认识做什么？如果你的"懂"没有改变你的行动能力，那可能只是感性层面的印象，而非理性的把握。 |
| 2 | 读书很多但用不上怎么办？ | 理论和实际统一，有的放矢 | 你最近读的书是针对哪个具体问题的？ |
| | | **扩展**：理论与实践脱节是认知中最常见的困境。毛选强调"有的放矢"——理论应该像箭一样对准实际问题的靶子。当你说"用不上"时，很可能是在为读书而读书，读之前没有带着一个真实的问题。没有问题的学习就像没有地图的旅行，知识无处附着。 |
| 3 | 如何判断自己的想法对不对？ | 实践是检验真理的唯一标准 | 你打算用什么客观标准来检验？ |
| | | **扩展**：人们常常用"我感觉""大家都这么说""书上写了的"来验证想法，但这些都不是客观标准。毛选认识论的核心主张是：一个想法对不对，最终只能靠实践的结果来判断。关键是找到可测量、可观察的检验标准——如果找不到，这个想法可能还没成型。 |
| 4 | 学了新知识记不住怎么办？ | 实践→认识→再实践循环 | 你学完之后 24 小时内做了什么实践？ |
| | | **扩展**：记忆问题本质上是循环断了——你只完成了"认识"这一半，没做"再实践"的后续。毛选的认知循环不是"学→记"，而是"做→学→再做→深化学"。信息只在被使用时才会留下痕迹。如果学完后24小时内没有任何实践动作，大脑会把这段信息标记为不重要的噪音而丢弃。 |
| 5 | 为什么对同一件事前后看法不同？ | 认识随实践发展而深化 | 这期间你的实践发生了什么变化？ |
| | | **扩展**：人们常把"看法改变"等同于"立场不坚定"，但毛选的认识论恰恰认为，看法变化是认识深化的正常标志。关键不在于看法变了没有，而在于**什么导致了变化**——是积累了新经验？发现了新事实？还是仅仅受到他人观点的影响？前者是认识的进步，后者只是盲从。 |
| 6 | 如何避免"想当然"？ | 没有调查就没有发言权 | 你下结论之前做过哪些调查？ |
| | | **扩展**："想当然"的本质是跳过调查直接下结论。毛选认识论对此的解法非常明确：发言权不是天生的，是用调查换来的。避免想当然不是靠"更小心"，而是靠建立"先调查后判断"的习惯。哪怕是一个小问题，问一句"我的判断有事实依据吗"就能防止大多数想当然。 |
| 7 | 别人的经验可以直接套用吗？ | 矛盾的特殊性 | 你的条件和别人的条件差在哪里？ |
| | | **扩展**：人们复制成功经验时常犯两个错误：要么因为条件不同而完全否定别人的经验，要么不管条件差异而全盘照搬。毛选的矛盾特殊性原理告诉我们——每一个成功方案都是特定条件（时间、地点、资源、人际关系）下的产物。值得学习的是方法背后的逻辑，而非具体做法本身。 |
| 8 | 怎样把零散经验变成系统认识？ | 去粗取精、去伪存真、由此及彼、由表及里 | 这些经验之间有什么共同点？ |
| | | **扩展**：这是毛选认识论中最精妙的方法论之一——十六字诀。零散经验就像拼图碎片，单独看没有意义。系统化的关键是四步：去掉偶然的、保留本质的（去粗取精）；排除假的、留下真的（去伪存真）；建立联系（由此及彼）；找到规律（由表及里）。大多数人只做到了第一步就停了。 |
| 9 | "直觉"可靠吗？ | 感性认识的局限性 | 你的直觉在过去被验证过几次？ |
| | | **扩展**：直觉是感性认识的最高形式，但它仍然是感性的——来自经验积累的快速模式匹配。毛选认识论承认直觉的价值（它是实践的产物），但不承认直觉的终极权威。关键在于：你的直觉是在什么条件下形成的？是经过大量实践反馈校正过的，还是建立在几个孤立案例上的？前者可靠，后者是偏见。 |
| 10 | 如何判断学习方向对不对？ | 实践检验，失败是成功之母 | 你的学习解决了什么实际问题？ |
| | | **扩展**：学习方向对错的唯一标准在实践中——它解决你当初想解决的问题吗？许多人学了很久才发现方向不对，原因是在学习之前没有明确"我要解决什么具体问题"。没有靶子，箭自然射不中。迷路时第一步不是加速走，而是确认方向。 |
| 11 | 看了很多方法论还是做不好？ | 实践是认识的目的，不是空谈 | 你完整执行过一个方法到出结果吗？ |
| | | **扩展**：方法论的泛滥是这个时代的独特困境——读过的框架越来越多，但做事的水平没有提升。毛选认识论的诊断很直接：你把认识当成了目的。方法论只是认识层面的工具，**真正的学习发生在执行中**。如果你没有把一个方法从第一步执行到最后一步直到产出结果，你其实还没有开始真正的学习。 |
| 12 | 理论和现实差距大怎么办？ | 理论需要在实践中修正和发展 | 是理论错了还是你没用对条件？ |
| | | **扩展**：理论和现实永远存在缝隙——理论是抽象的、理想化的，现实是具体的、充满变数的。毛选认识论的关键洞察是：**理论服务于实践，而不是反过来**。当差距出现时，需要判断的两个方向是：理论本身需要修正（它可能过时了或不适用），或者你没有正确地应用它（忽略了关键条件）。通常两者都有。 |
