# 类别 B：困境与选择（辩证法）

- **种子问题数**：15
- **对应哲学支柱**：辩证法（矛盾分析、转化与结构）
- **适用场景**：决策困难、复杂局面梳理、两难选择、风险评估、计划失控

---

## 设计说明

困境与选择是辩证法在日常生活中最密集使用的领域——当局面复杂、选项冲突、不确定性高时，人们最需要矛盾分析工具。此类种子问题的核心逻辑是：**用辩证法把"感觉上的困境"转化为"结构上的矛盾"**，让用户从被动的情绪反应转向主动的矛盾分析。每一道问题都是"抓主要矛盾"方法论的不同切面。

---

## 种子问题表

| # | 种子问题 | 核心原理 | 追问方向 |
|---|---------|---------|---------|
| 1 | 复杂局面无从下手？ | 抓主要矛盾 | 当前困住你的最关键的那个问题是什么？ |
| | | **扩展**：复杂局面的典型特征是"所有问题同时冒出来"，让人陷入应激反应。毛选辩证法对此的判断是：在一个复杂事物中，总有一个矛盾是主要的、决定性的，它规定了其他矛盾的地位。如果你觉得每件事都很急，说明你还没有找到那个"牵一发而动全身"的关节点。找到它，80%的混乱会自然消退。 |
| 2 | 两件事都很重要先做哪个？ | 主要矛盾 vs 次要矛盾 | 哪个不解决，另一个也无法进行？ |
| | | **扩展**：两个选择都很重要，恰恰说明它们的重要性不是一个层面的。毛选辩证法区分主次矛盾的方法很实用：看排除关系。如果A不解决，B就无法推进，那么A就是主要矛盾。反之亦然。如果两者互不依赖，那可能它们各自是不同系统的主矛盾，需要分别处理，而不是在一个系统里排序。 |
| 3 | 努力了还是失败？ | 内因外因关系 | 失败的原因中哪些是你自己能控制的？ |
| | | **扩展**：毛选的"内因是根据，外因是条件"是一个极其实用的归因框架。失败后最常见的情绪反应是全部归因于外部（运气、市场、他人），或者全部归因于内部（能力差、不够努力）。两者都是片面的。正确的分析路径是：**区分什么是你可以改变的，什么是你不能改变的**，然后把你能做的部分做到极致。 |
| 4 | 该不该听别人的批评？ | 矛盾的主要方面，兼听则明 | 批评者了解全部情况吗？他的依据是什么？ |
| | | **扩展**：面对批评，人们容易走两个极端——全盘接受或全部否定。毛选辩证法的态度是：**具体分析批评的立场和依据**。批评者的信息是否完整？他的出发点是什么？他的判断中哪些有事实依据、哪些只是推测？批评的价值不在于声音大小，而在于它能否帮你看到矛盾的另一个方面——你可能是矛盾的次要方面。 |
| 5 | 坏事能变成好事吗？ | 矛盾的同一性（转化条件） | 需要什么条件才能转化？ |
| | | **扩展**：这是毛选辩证法中最富乐观精神的核心原理——矛盾双方在一定条件下互相转化。坏事不是自动变好事的，**转化需要条件**。这个问题的价值在于帮助用户从情绪反应转向条件分析：我需要创造什么条件才能让局面反转？转化条件越具体，从"坏事"思维转向"好事"就越有路径。 |
| 6 | 怎么评估一个选择的风险？ | 矛盾双方的力量对比 | 最坏的结果是什么？概率多大？你能承受吗？ |
| | | **扩展**：风险评估的本质是分析矛盾双方的力量对比——对你有力的因素和不利的因素分别有多强。毛选辩证法反对主观臆断的风险判断，主张具体分析每个因素的实际分量。最有用的三个问题：最坏结果是什么？它发生的可能性多大？如果发生了，你能不能兜底？回答了这三个问题，95%的风险焦虑会消失。 |
| 7 | 事情一团糟怎么理出头绪？ | 分析矛盾结构 | 把所有问题列出来，哪个是根源？ |
| | | **扩展**：混乱不是因为事情多，而是因为没有结构。毛选辩证法处理混乱的方法论很简单：**先把所有问题摆到桌面上**（列清单），然后分析它们之间的因果关系。问题A是问题B的原因，还是B是A的原因？找到了那个作为原因最多的节点，你就找到了混乱的"钥匙"。 |
| 8 | 做决定时总是犹豫不决？ | 研究矛盾各方面的特点 | 你在等什么信息？这些信息能获得吗？ |
| | | **扩展**：犹豫不决通常有两个来源：信息不足或信息过载。毛选辩证法对此的处方是：你不需要知道全部信息才做决定，你只需要知道矛盾各方面的**核心特征**。花时间搞清楚每一方的底线、优势、弱点，比收集海量数据更有用。如果等到100%确定才行动，你其实永远无法行动。 |
| 9 | 两个方案各有利弊怎么选？ | 矛盾的主要方面 | 哪个方案的缺点是可以接受的？ |
| | | **扩展**：当两个方案各有利弊时，传统"利弊分析法"往往无效——因为利弊在同一层面，无法取舍。毛选辩证法的方法更有穿透力：**看哪个方案的主要矛盾方面是可接受的**。没有一个方案是完美无缺的，关键是问自己——哪个方案的缺点我能承受？这个"能承受的缺点"就是矛盾的主要方面，它决定了选择方向。 |
| 10 | 怎样判断事情还有没有转机？ | 转化条件的分析 | 促成好转的条件是否在增加？ |
| | | **扩展**：判断一件事有没有转机，不是靠心态（"我相信会好起来"），而是靠对转化条件的客观分析。毛选辩证法的方法是：**列出有利于转化的条件清单和不利于转化的条件清单**，然后看趋势。如果有利条件在增加（哪怕速度很慢），转机就在酝酿中。如果不利条件在固化，那就要考虑调整策略甚至止损。 |
| 11 | 计划总赶不上变化怎么办？ | 矛盾发展的阶段性 | 变化来自外部条件还是内部判断？ |
| | | **扩展**：变化是绝对的，不变是相对的。计划"赶不上"变化，通常是因为计划本身是线性的、静态的，没有考虑事物发展的阶段性。毛选辩证法认为，每一个矛盾在发展的不同阶段有不同的特征和规律。**关键不是让计划不变，而是让计划具备阶段性调整的能力**。先判断变化来自哪——外部条件变了，还是你之前对情况的判断出了偏差——再决定是调整计划还是重新制定。 |
| 12 | 什么时候该坚持，什么时候该放弃？ | 主次矛盾转化，量变质变 | 坚持的条件还在不在？放弃的损失能否承受？ |
| | | **扩展**：这是辩证法中最经典的实用问题。毛选对此没有简单的是或否，而是给出了两个分析维度：(1)主要矛盾是否发生了转化——你为之坚持的那个核心问题是否还存在？(2)量变是否接近质变——你已付出的努力是否已经积累了足够的基础，放弃意味着前功尽弃？回答这两个问题，比"坚持就是胜利"式的口号有用得多。 |
| 13 | 信息不足怎么做决定？ | 先调查后决策（反对本本主义） | 最快能获得关键信息的方法是什么？ |
| | | **扩展**：信息不足时，最危险的策略是"凭经验猜"或者"等所有信息到齐"。毛选的"反对本本主义"提出了第三条路：**用最快的速度获得最少但最关键的信息**。不需要全面调查，只需要针对决策的核心变量做定向调查。问自己：如果我只能问一个人或查一个数据，哪个信息对我决策的帮助最大？找到了，就去做，不等。 |
| 14 | 越急越出错怎么办？ | 反对主观主义、表面性 | 你是想要速度，还是想要效果？ |
| | | **扩展**：急和出错之间的恶性循环，根源在于**主观主义**——把自己的焦急当成了客观的紧迫性。毛选辩证法把这叫作"表面性"：只看问题的表象（时间紧、压力大），没看到问题的本质（速度不是目的，效果才是）。停下来问自己：我真正的目标是按时完成，还是交出一个有效的结果？如果效果不对，速度就是白费。 |
| 15 | 被领导针对但不敢辞职（因为房贷/家庭），很压抑？ | 内因外因关系，可能性与现实性 | 当前困境中哪些因素是你可控制的，哪些不可控？ |
| | | **扩展**：这种困境之所以痛苦，是因为你同时被"外因"（领导的行为）和"内因"（你的恐惧）夹击，而且觉得两条路都走不通——留下来痛苦，离开又不敢。毛泽东的"内因外因关系"提供了一个出口：外因（领导）你控制不了，但内因（你的恐惧）你可以处理。恐惧的核心是什么——是不确定离开后的生活，还是对自己能力的怀疑？把"最坏的可能性"分析清楚（L16274），划清你的底线。一旦最坏结果能承受，恐惧就失去了一半的力量。同时用"可能性与现实性"的框架拆解两条路：留下来有没有改善的路径？离开需要准备什么条件？当你有了一二三步计划，压抑就开始转化为行动。 | 留在这个岗位的改善路径是什么？离开需要准备哪些条件？ |
