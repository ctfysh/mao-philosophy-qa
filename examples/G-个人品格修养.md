# 个人品格修养——感觉自己在"混日子"怎么办？

> 老赵在国企工作七年，同一岗位、同一流程、同一群人。每天早上要在车里坐一会儿才愿意进办公室——不是累，是"没劲"。

```
用户：每天做的事跟昨天一样，跟去年也一样。领导说我稳当，同事说我靠谱，但我知道我就是在混。我有时候想，这辈子就这样了？
```

**毛选原理**：放下包袱，开动机器（《学习和时局》L7776）。"所谓放下包袱，就是解除精神上的许多不必要的负担。所谓开动机器，就是要善于使用思想器官。"

**分析**：你说"混日子"，但你七年没换工作、没出大错，说明你不是懒。你"没劲"是因为你的能力已经远超工作需求，没有了挑战对象。混日子不是因为你不想干，是因为没有"需要斗争的事情"。你觉得困住你的是环境，但困住你的可能是一个想法——"我该做的就这些了"。

**行动指引**：这个星期做一件事——不改变工作内容，只改变观察方式。去看看单位的哪些环节效率低、哪些流程可以优化、哪些年轻人需要指导。

```
┃ ▼ 继续追问 ┃
你最近一次主动去解决一个困难的问题，是什么时候？
```

> 老赵想了很久，上一次可能是三年前单位上新系统的时候。

```
用户：上次主动解决问题是三年以前了，上了一个新系统，我花了很大功夫去学。
```

**毛选原理**：工作就是斗争（《关于重庆谈判》L9218）。"什么叫工作，工作就是斗争。那些地方有困难、有问题，需要我们去解决。我们是为着解决困难去工作、去斗争的。"

**分析**：你三年前觉得"有劲"，是因为面对了新系统这个"斗争对象"。系统稳定后斗争对象消失了，但你的工作能力还在。你不是只能"混"，而是你以为"混"是唯一的选择。你单位有没有问题？办公流程、年轻人带教、政策落地——这些都是你经验能够得着的地方。

**行动指引**：下周一主动去找新来的两个年轻人聊聊，问问他们工作上有什么困难，你可以帮上什么。这不叫"出风头"，叫"传帮带"。

```
┃ ▼ 继续追问 ┃
你担心"做多了别人说爱出风头"——这是事实，还是你给自己背的包袱？
```

> 老赵开始每周和年轻人聊聊，后来领导知道了，正式让他做了新人的指导人。

```
用户：我每周找个年轻人聊聊天帮帮忙，后来领导知道了，让我正式当了新人的指导人。我现在上班不是"去混"了，是真的有事做。
```

**毛选原理**：放下包袱，开动机器——完整验证。放下"怕人说"的包袱，机器的启动只需要一个小的开始。

**分析**：环境没变，单位没变，但你的状态变了——因为"你换了一种方式对待同一个地方"。你把边界画大了一点，工作就回来了。这个经验可以用在任何一段让你感到停滞的关系或位置中：不是环境困住你，是你给自己的边界太小了。

**行动指引**：每个月问自己一次：我现在的边界是自己画的，还是环境给的？

```
┃ ✓ 追问总结 ┃
1. 核心矛盾：不是"混日子"，是"没有了斗争对象"
2. 行动方向：把边界画大一点，走出职责范围寻找可以解决的问题
3. 以后可先问自己：我现在的边界是自己画的，还是环境给的？
```
