# 人际关系——闹矛盾了怎么办？

> 小周和认识八年的朋友合伙开工作室，半年后因为分工和利益分配问题吵了三次。最近一次争吵后，朋友两周没联系他，工作室的工作也搁置了。

```
用户：八年了，从来没红过脸。合伙之前也想过会有矛盾，但没想到会闹成这样。他现在不理我，我也不敢主动找——怕主动了就输了。
```

**毛选原理**：团结—批评—团结（《关于正确处理人民内部矛盾的问题》L14585）。"从团结的愿望出发，经过批评使矛盾得到解决，从而在新的基础上达到新的团结。"

**分析**：注意顺序——从团结出发。你现在卡在"先分对错还是先修复关系"这个问题上。"怕主动就输了"这个想法，假设了你们的关系是一场输赢游戏。但如果你想要的是修复关系、工作室恢复运转，那"主动"不是认输，而是第一步。

**行动指引**：不谈分歧，先恢复连接。给他发一条消息，不提吵架、不提工作室，就说："最近工作室没动静，有点想你了。周末出来吃个饭？" 他不会拒绝。

```
┃ ▼ 继续追问 ┃
你想要的最终结果是什么？是让他认错，还是工作室恢复、关系也恢复？
```

> 小周说想要关系恢复，但咽不下那口气。

```
用户：我当然想要和好。但我又咽不下这口气——他也有问题，凭什么我先低头？
```

**毛选原理**：区分两类不同性质的矛盾（《关于正确处理人民内部矛盾的问题》L14568）。对抗性矛盾（敌我）和非对抗性矛盾（人民内部）处理方法完全不同。

**分析**：你们是目标一致（把工作室做好）但方法不同（分工和利益分配方式有分歧）。这是典型的非对抗性矛盾，也就是说不需要"赢"，只需要找到一个双方都能接受的方案。你咽不下的是"面子问题"，但如果把这个放到"工作室能不能继续"面前比一下，哪个分量更重？

**行动指引**：吃饭的时候先聊天，恢复朋友之间的感觉。如果气氛好，可以说一句："之前的事我想了很多，我们的目标是一样的，就是方式需要调整。要不重新聊一下分工？"

```
┃ ▼ 继续追问 ┃
你愿意先迈出这一步，还是宁愿等对方先开口？
```

> 小周发了消息，朋友秒回了。两人吃饭后长谈了一次，重新调整了分工和分成方式。

```
用户：我发了消息，他秒回了。周末一起吃了饭，刚开始有点尴尬，但聊开了就好了。后来我们重新谈了分工和分成，现在工作室恢复了，关系反而比以前更好了。
```

**毛选原理**：团结—批评—团结的完整过程——在新的基础上达到了新的团结。经过了矛盾考验的关系，比回避矛盾的关系更牢固。

**分析**：你亲自验证了这个公式的有效性。第一步主动不是认输，是"从团结出发"；中间谈分工不是吵架，是"开展批评"；最后达成的新方案是"新的团结"。

**行动指引**：在合作中建立一个定期沟通机制——每周花 30 分钟聊一下分工配合有什么不舒服的地方，不等矛盾积累到爆发。

```
┃ ✓ 追问总结 ┃
1. 核心矛盾：想要修复关系，但被"谁先认输"的心理卡住了
2. 行动方向：从团结出发（主动恢复连接），再谈分歧（调整方案）
3. 以后可先问自己：我是想赢，还是想解决问题？
```
