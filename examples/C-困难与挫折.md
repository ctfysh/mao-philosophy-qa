# 困难与挫折——困难太大感觉撑不下去了？

> 小马考研两年都失败了。第一年差 15 分，第二年差 2 分。家里劝他放弃了，他自己也开始怀疑。

```
用户：两年了，每天早出晚归，结果就差这么一点点。有时候半夜醒来就在想，是不是我确实不行？
```

**毛选原理**：内因是根据，外因是条件（《矛盾论》L3130）。外因通过内因而起作用。

**分析**：你说"差一点"，我们先看看这两个"差一点"是不是同一个问题。第一年英语差 5 分过线，第二年你把英语从 50 提到了 65，但专业课从 120 掉到了 105。这不是"我这个人不行"，而是"时间资源没分配对"。这是两个完全不同的归因——前者让你绝望，后者可以优化。

**行动指引**：把两年各科的分数列出来，看看到底是"能力上限"的问题还是"精力分配"的问题。

```
┃ ▼ 继续追问 ┃
第二年你花在英语上的时间，和第一年比多了多少？专业课又少了多少？
```

> 小马仔细算了一下：第二年他几乎把 60% 的复习时间给了英语，专业课全靠刷真题维持。

```
用户：第二年英语花了差不多 60% 的时间，从 50 提到 65。专业课从之前的 120 掉到 105。确实是把时间抽走了。
```

**毛选原理**：矛盾的主要方面（《矛盾论》L3252）。抓住了主要矛盾的主要方面，问题就迎刃而解。

**分析**：你现在面临一个资源分配的优化问题。英语从 50 到 65 花了巨大代价（60% 的时间），但从总分角度看，英语提 15 分和专业课降 15 分，净效果是零。如果你的目标是**总分过线**，那专业课是主要矛盾（150 分的大头），英语稳住 60 分即可。你之前的策略是"补短板"，但更有效的可能是"强长板"。

**行动指引**：如果英语目标从 65 降到 60，每天只需 1 小时，剩下时间全给专业课。这样专业课恢复到 120 以上，总分就能过线。

```
┃ ▼ 继续追问 ┃
英语降到 60 分的目标，你能接受吗？还是会觉得"好不容易提上来了，降回去可惜"？
```

> 小马虽然对降英语目标有些不舍，但理性上接受了这个方案。

```
用户：接受。英语 60 够了，剩下的精力放在专业课上。但我还是有点怕——万一再来一次又差两分呢？
```

**毛选原理**：认识总公式——"实践、认识、再实践、再认识"（《实践论》L3110）。每一循环的内容，都比较地进到了高一级的程度。

**分析**：你第一年发现英语是短板 → 第二年补英语 → 发现顾此失彼 → 现在调整策略。这不是"失败了两次"，而是"完成了两次认识循环，获得了第三年的正确策略"。每一轮失败都不是白费的，它让你对这场考试的理解更深一层。

**行动指引**：制定一个每周有检查点的复习计划。每周检查时间分配是否合理，随时微调。

```
┃ ✓ 追问总结 ┃
1. 核心矛盾：不是能力问题，是精力分配问题
2. 行动方向：保专业课（主要矛盾），英语稳 60 即可
3. 以后可先问自己：之前的失败告诉我什么新信息？我的策略需要怎么调整？
```
