# 自我成长——怎么才能坚持下去？

> 小刘每次立 flag——学英语、健身、早起——最多坚持三周就放弃。他给自己贴了标签："我就是三分钟热度的人。"

```
用户：今年我立了 8 个 flag，全倒了。我在想要不要干脆放弃立 flag——反正也做不到。
```

**毛选原理**：调查研究（《反对本本主义》L1685）。"一切结论产生于调查情况的末尾，而不是在它的先头。"

**分析**：在得出"我就是三分钟热度"这个结论之前，先做一次调查。今年 8 次尝试里，有没有哪一次坚持得最久？那次和其他几次有什么不同？你可能会发现，问题不是"我没有意志力"，而是"某些条件下我能坚持，某些条件下我不能"。

**行动指引**：列一张表，把 8 次尝试写下来，分别写上坚持了多久、当时有什么外部条件（有人监督吗？有固定时间吗？有付费吗？）。

```
┃ ▼ 继续追问 ┃
8 次里最长的那次是什么？它有什么特殊条件？
```

> 小刘发现坚持最久的是学英语（5 周），那次他报了一个班，每周固定时间上课。

```
用户：最长的是学英语那次，坚持了 5 周。那次报了一个班，每周二四晚上七点有课，老师已经在了，我不好意思不去。
```

**毛选原理**：感性认识→理性认识（《实践论》L3050）。"感觉到了的东西我们不能立刻理解它，只有理解了的东西才能更深刻地感觉它。"

**分析**：你之前的感觉是"我就是三分钟热度"（感性认识）。现在调查完了，你发现了一个模式：**有固定时间节点和外部约束时，你能坚持；全靠意志力时，你坚持不了。** 这意味着你的问题不是性格缺陷，而是**缺少一个适合你的坚持机制**。

**行动指引**：如果这次再学英语，不报班，能不能自己设计一个同样有"固定时间+外部约束"的机制？比如找一个学习搭子，每天晚 8 点视频连线学 30 分钟，不来发 50 红包。

```
┃ ▼ 继续追问 ┃
每天连线 30 分钟，这个频率在你的生活中能执行的概率有多大？
```

> 小刘将方案调整为每周 5 次，周日提前约好本周的时间。

```
用户：每天可能有点难，有时加班。改成每周至少 5 次，每周日提前约好时间，没达到就发红包。
```

**毛选原理**：认识总公式——"实践、认识、再实践、再认识"（《实践论》L3110）。循环往复以至无穷。

**分析**：你看，你从"我不行"到了"我需要调整频率"——这是完全不同的心态。你不是在否定自己，而是在优化方案。这个方案在实践中可能还会遇到问题（搭子也放弃了、某段时间太忙），但这不意味着你又"不行"了，只是方案需要再调整。

**行动指引**：今晚就找一个也想学英语的朋友，约好规则。第一周结束后复盘，看方案需不需要调整。

```
┃ ▼ 继续追问 ┃
你能接受"方案可能需要调整"这个前提吗？还是觉得"调整了就是失败了"？
```

> 一个月后小刘打卡 22 天，中间忙的时候调整为每周 4 次，但没断过。

```
用户：一个月了，打了 22 天卡。中间有两周特别忙，改成每周 4 次，但没有断过。
```

**毛选原理**：改造主观世界（《实践论》L3084）。学习的本质是改造自己的认知方式。

**分析**：你最大的变化不是英语进步了多少，而是你关于"坚持"的认知被改写了——从"坚持是意志力问题"变成了"坚持是设计问题"。这个认知可以迁移到你想做的任何事情上。

**行动指引**：把这次的经验写成你的"坚持方法论"：遇到想坚持的事，先问自己"我需要什么机制"。

```
┃ ✓ 追问总结 ┃
1. 核心矛盾：把"性格问题"（我没意志力）重新定义为"方法问题"（我需要机制）
2. 行动方向：固定时间节点 + 外部约束（搭子/红包）
3. 以后可先问自己：我需要一个什么样的机制来帮助我坚持？
```
