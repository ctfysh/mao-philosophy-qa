# 毛选哲学问对 — 用毛泽东哲学方法论分析你的人生困惑

> 这不是一个"答案机器"，而是一个**思维教练**。
> 不替你决策，而是用《实践论》《矛盾论》《反对本本主义》中的方法论，帮你自己想清楚。

---

## 快速开始

在对话中输入以下任一命令启动：

| 命令 | 说明 |
|------|------|
| `/mao-qa` | 主命令 |

如果当前对话涉及"矛盾""实践""实事求是""调查""方法"等关键词，AI 也会自动推荐。

## 怎么用

**直接说你遇到的问题就行。** 比如：

> "刚毕业，不知道该选喜欢的工作还是稳定的工作。"
> "跟同事合作总被拖进度，说了也没用。"
> "坚持健身三个月了感觉没变化，还要继续吗？"
> "朋友总找我借钱，不借怕伤感情，借了又不知道怎么办。"
> "每天都在忙但没什么成果，不知道问题出在哪。"

AI 不会直接告诉你"该怎么做"——它会追问你，用哲学框架帮你把问题拆开，让你自己看到答案。

**如果你还没想好问什么**，AI 会给你一个菜单分类引导你选方向。

## 对话过程是什么样的

典型的对话长这样：

```
你：刚毕业，不知道该选喜欢的工作还是稳定的工作。

AI：[引用《矛盾论》抓主要矛盾的原理]
   → 追问：这两个选择里，哪个放弃了你会更后悔？
```

```
你：我不知道，都很重要。

AI：[引用《反对本本主义》调查研究的原理]
   → 追问：你对这两个选择背后的真实生活了解多少？
   比如，喜欢的工作实际每天做什么？稳定的工作天花板在哪？
```

```
你：喜欢的工作了解一些，但不知道是不是自己想的那样……

AI：[引用《实践论》从战争中学习战争的原理]
   → 追问：你有没有办法先试一段时间低成本体验一下？
```

整个过程 AI 只做三件事：
1. **承** — 肯定你的处境，让你感觉被理解
2. **转** — 引入一条哲学原理，帮你换个视角看问题
3. **问** — 用一个开放式问题引导你继续深入

## 跟普通 AI 问答有什么不同

| 普通 AI | 毛选问对 |
|---------|---------|
| 直接给你答案 | 追问让你自己得出结论 |
| 引用各种来源，权威感强 | 只锚定原文线号，方法论导向 |
| 一次性把所有信息给你 | 每轮只给 1 条原理，逐步深挖 |
| 你说什么就回答什么 | 会检测你的情绪状态和追问深度 |
| 对话结束就结束 | 下次来可以接着上次聊 |

## 三种模式

| 模式 | 什么时候用 | 什么感觉 |
|------|-----------|---------|
| **引导模式**（默认） | 有困惑，但不清楚问题在哪 | AI 一步步追问，帮你理清思路 |
| **教学模式** | 想学哲学方法论本身 | AI 先讲原理再举例，最后让你用在自己的事上 |
| **顾问模式** | 已经想清楚了，想看看有没有盲点 | AI 用矛盾论检验你的逻辑，指出可能遗漏的角度 |

## 常见问题

**Q：我不会哲学，能用吗？**
A：你不需要懂任何哲学。AI 负责把它翻译成日常语言，你只需要说自己的事。

**Q：说的不对怎么办？**
A：直接说"不对"或"不是这样"——AI 会调整追问方向。你的反馈本身就是认识过程的一部分。

**Q：可以问关于政治的问题吗？**
A：这个工具只提取方法论核心（认识论、辩证法、方法论），不涉及政治立场或历史评价。

**Q：一次可以问多个问题吗？**
A：建议一次只问一个。如果你说了多个问题，AI 会帮你挑出最核心的那个先分析。

**Q：想要直接答案怎么办？**
A：AI 不会给直接答案——不是因为它不知道，而是因为它给的"答案"大概率不适用你的具体情况。它会给你分析框架，结论由你自己得出。如果你试了觉得这种方式不适合你，随时可以退出。

## 哲学基础（如果你想了解来源）

所有的追问和分析都基于《毛泽东选集 1-7》中的哲学方法论部分，提取为四大支柱：

- **认识论**（实践与认识的辩证关系）—— 怎么认识世界和自己
- **辩证法**（矛盾分析法）—— 怎么分析问题中的对立和转化
- **方法论**（调查研究、工作方法）—— 怎么行动和解决问题
- **人生观与战略**（品格修养、公私关系）—— 该以什么姿态面对人生

每条分析都锚定原文具体行号，可追溯可验证。
